Group1:

1. Установить отношения One-to-many между Client и Account
2. Реализовать REST API вывода статистики по всем счетам клиента.
В статистику включить:
- общий суммарный баланс в долларах
- Map общего баланса счетов в разных валютах

	URL: /client/summary
	Метод: GET

	Ожидаемые HTTP статусы: 200, 404

Group2:
1. Установить отношения One-to-many между Account и Transaction
2. Реализовать REST API вывода статистики по всем входящим и исходящим переводам счета.
В статистику включить:
- количество входящих и исходящих переводов
- суммарные расходы в долларах
- суммарные доходы в долларах

	URL: /account/summary
	Метод: GET

	Ожидаемые HTTP статусы: 200, 404
